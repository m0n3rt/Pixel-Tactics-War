# 像素战棋 PIXEL TACTICS

基于 Canvas 的像素风自动战棋游戏，纯 HTML / CSS / JS 实现，无需任何框架或依赖。

---

## 游戏概览

- **模式**：PVE — 玩家 vs 简单 AI
- **流程**：首页 → 加载过渡 → 回合提示横幅 → 准备阶段（布阵 60s） → 自动战斗 → 回合结算 → 循环 / 游戏结束
- **胜利条件**：摧毁敌方总部（HP 归零）

---

## 核心参数

| 参数 | 值 |
|---|---|
| 棋盘尺寸 | 14 列 × 8 行（32px 像素格） |
| 总部区域 | 两侧各 2 列（画布内渲染城堡） |
| 战场区域 | 中间 10 列（列索引 2 ~ 11） |
| 玩家布阵区 | 左侧 3 列（列索引 2 ~ 4） |
| 敌方布阵区 | 右侧 3 列（列索引 9 ~ 11，AI 随机生成） |
| 总部初始 HP | 100 |
| 准备时间 | 60 秒 |
| 每回合布阵点数 | 20 |

---

## 单位类型

| 兵种 | 字符 | 花费 | HP | 攻击 | 射程 | 移动CD | 攻击CD | 形状 |
|---|---|---|---|---|---|---|---|---|
| 步兵 infantry | 步 | 1 | 12 | 3 | 1 | 0.6s | 0.9s | 方块 |
| 弓手 archer | 弓 | 2 | 8 | 3 | 3 | 0.8s | 1.2s | 菱形 |
| 重甲 tank | 甲 | 3 | 25 | 4 | 1 | 1.0s | 1.0s | 双层方块 |

- 每个单位拥有**独立的移动 / 攻击冷却计时器**。
- 不在射程时，单位按移动间隔向最近敌人靠近（优先水平方向，主方向被堵则尝试另一轴）。
- 进入射程后按攻击间隔自动攻击。
- **同一格不允许重叠**（防碰撞移动系统）。

---

## 战斗规则

1. **回合结算**：一方场上单位全灭时，本回合结束。
2. **总部伤害**：胜利方剩余单位的 **总最大生命值之和 × 50%**，对敌方总部造成伤害。
3. **HP 钳位**：总部 HP 不会低于 0，UI 数值和血条均已钳位。
4. **总部摧毁**：HP 归零时触发爆炸特效（大量火焰 + 烟雾粒子），城堡变为废墟状态（残垣断壁 + 持续冒烟动画 + 倒下旗杆 + "DESTROYED" 标签）。
5. **游戏结束**：任意一方总部 HP 归零 → 显示"再来一局" / "主界面"按钮。

## 界面与交互

### 游戏流程画面

| 画面 | 说明 |
|---|---|
| 首页 | 标题渐变动画 + 剑图标浮动 + "开始游戏"按钮 |
| 加载过渡 | 旋转 spinner + "正在准备战场..."，1.2 秒 |
| 回合横幅 | "第 N 回合" 弹出缩放动画，1.8 秒 |
| 准备阶段 | 棋盘 + 布阵操作 + 60 秒倒计时 |
| 自动战斗 | 单位自动移动 / 攻击 + 粒子特效 |
| 回合结算 | 战果信息 + "下一局"按钮 |
| 游戏结束 | "再来一局" / "主界面" |

### 顶部状态栏

- **左侧**：我方总部 HP 面板（数值 + 绿色渐变血条）
- **右侧**：敌方总部 HP 面板（数值 + 红色渐变血条）
- **中间**：阶段徽章（准备阶段 / 战斗中 / 结算，配色各异）、回合数、倒计时、布阵点数、实时军队摘要（步兵×N 弓手×N 重甲×N）

### 画布渲染

- **总部城堡**：像素风城堡建筑，随 HP 降低逐步破损：
  - HP > 70%：完好状态
  - HP 40~70%：出现裂痕
  - HP 15~40%：城垛脱落 + 大量裂痕 + 碎石
  - HP = 0：废墟状态（残垣断壁 + 焦痕地面 + 持续冒烟 + 倒下旗杆 + DESTROYED 标签）
- **飘动旗帜**：城堡顶部动态旗帜（正弦波飘动）
- **受击闪光**：总部被打时红色闪烁叠加
- **单位渲染**：各兵种独立形状 + 类型汉字（步/弓/甲）+ 阵营边框（绿=我方/红=敌方）+ 血条
- **部署区域**：准备阶段己方区域脉冲高亮 + "我方区域"/"敌方区域"文字标签
- **悬浮高亮**：鼠标悬浮格子边框高亮

### 右侧战斗日志

- 画布右侧 180px 长条面板，与画布等高
- 实时滚动显示战斗记录，最多保留 60 条
- 分色显示：攻击（红色）/ 击破（橙色）/ 系统信息（蓝色）

### 粒子系统

| 粒子类型 | 用途 |
|---|---|
| `proj`（投射物） | 远程攻击弹道轨迹 |
| `spark`（火花） | 单位死亡碎片、总部受击特效、爆炸火焰 |
| `smoke`（烟雾） | 总部摧毁后的爆炸烟雾，上浮淡出 |

### 单位悬浮提示

鼠标悬浮到任意单位上时，显示浮动信息卡：
- 阵营（我方 / 敌方，绿 / 红色标注）
- 兵种名称、HP / 最大HP、攻击、射程、移动CD、攻击CD

## 操作说明

### 鼠标操作

| 操作 | 效果 |
|---|---|
| 点击布阵区空格 | 放置当前选中兵种（消耗点数） |
| 点击已有己方单位 | 移除该单位并返还点数 |
| 鼠标悬浮单位 | 显示详细属性浮动提示 |
| 点击底部兵种按钮 | 切换选中兵种 |

### 键盘快捷键

| 按键 | 阶段 | 效果 |
|---|---|---|
| `1` / `2` / `3` | 准备（未暂停） | 选择 步兵 / 弓手 / 重甲 |
| `Space` / `Enter` | 准备（未暂停） | 立即开战 |
| `P` | 准备 + 战斗 | 暂停 / 继续 |
| `+` / `=` | 战斗 | 切换战斗倍速 1× / 2× / 3× |
| `Space` / `Enter` | 结算 | 进入下一局 |

### 暂停系统

- **准备阶段**和**战斗阶段**均可暂停（按钮 + 键盘 `P`）
- 暂停时：倒计时冻结、战斗冻结、**禁止部署棋子**
- 暂停弹窗（圆角卡片 + 毛玻璃背景）包含：
  - **▶ 继续游戏** — 恢复游戏
  - **✕ 退出游戏** — 弹出二次确认对话框（"确认退出" / "取消"），确认后返回主界面

### 战斗变速

- 战斗阶段底栏可切换 **1× / 2× / 3×** 倍速

## 运行方式

### 直接用浏览器打开（推荐）

无需任何依赖，双击 `index.html` 即可。

也可以在 VS Code 中右键 `index.html` → "Open With Live Server"。

### 使用本地 HTTP 服务器（可选）

```pwsh
cd pixel-tactics
npm install
npm run serve
```
